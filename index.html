Подробное описание товара/услуги

CloudDoc — облачный текстовый редактор с совместным редактированием в реальном времени. Платформа позволяет создавать, редактировать и комментировать документы одновременно с коллегами. Встроенная история версий, шифрование AES-256, API для интеграций с CRM и корпоративными мессенджерами. Доступен на всех устройствах: Web, iOS, Android.

Ценовой сегмент предложения

Средний ценовой сегмент (B2B SaaS). Бесплатный тариф для частного использования, Профессиональный — 19,99 BYN/мес, Бизнес — 49,99 BYN/мес. Корпоративные цены индивидуально.
